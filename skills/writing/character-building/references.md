# Sources

Personal notes on what this skill was built from. Not copied when the skill is
imported into a project — this is a memory-jog for the author, not part of the skill.

The skill distills these sources; it does not reproduce them. Follow the links /
titles below to the originals.

| Source | What it contributed |
|---|---|
| John Truby, *The Anatomy of Story* (ISBN 978-0865479937) | The **character web**: build characters by contrast, not in isolation ("a character is defined by who he is not"); the roles — hero, opponent, ally, fake-ally opponent, fake-opponent ally, subplot character; the hero–opponent relationship as the story's most important; the subplot character as a parallel problem resolved differently. |
| Robert McKee, *Story* (ISBN 978-0060391683) | **Characterization vs. true character** — true character is revealed by choices under pressure; structure and character are the same thing; complexity should fit the genre. |
| David Brown & Michelle Barker, *Immersion and Emotion* (ISBN 978-1738076611) | Show character through action/choice, not exposition; **no one is 100% anything** (Mary-Sue / Snidely Whiplash); the **hero's dark parallel** (Holmes/Moriarty); talents + flaws; everyone has a fear, a secret, a story (the iceberg); the memorable physical "wart"; relationship arcs as subplots. |
| Will Storr, *The Science of Storytelling* (ISBN 978-1419743030) | The **want (conscious) vs. need (subconscious) contradiction** as the mark of a multidimensional character (Lester in *American Beauty*); dialogue as compressed, subtext-dense character revelation. |
| Ross Hartmann & Esther Chilton, *The Structure of Story* (ISBN 978-1735603810) | The optional deeper vocabulary: **void, spine, ghost (trauma), lie/misbelief, worldview, obsession (deprival super-reaction), comfort concept, façade, individuation**; character as the embodiment of theme. |
| Jason Georgiades & Rosita Grigaite, *7 Day Screenplay* (Udemy) — udemy.com/course/sevendayscreenplay | The practical scaffolding questionnaire: who / what's special / want / need (central flaw) / why / method / obstacle / stakes; the likability levers. |

This skill leads with want-vs-need + ghost + lie + flaw/fear/secret (accessible
core) and offers Hartmann's fuller vocabulary as an optional deeper layer. It
branches at the start between building one character and mapping a cast web
(Truby). Pairs with `writing/character-audit`.
