# Sources

Personal notes on what this skill was built from. Not copied when the skill is
imported into a project — this is a memory-jog for the author, not part of the skill.

The skill distills these sources; it does not reproduce them. Follow the links /
titles below to the originals.

| Source | What it contributed |
|---|---|
| Will Storr, *The Science of Storytelling* (ISBN 978-1419743030) | Category 1: **want without need** — the want/need contradiction is what makes a character multidimensional. Category 7: dialogue must carry subtext. |
| David Brown & Michelle Barker, *Immersion and Emotion* (ISBN 978-1738076611) | Category 2 (**Mary-Sue / Snidely Whiplash** — no one is 100% anything); category 3 (the hero's **dark parallel**); category 4 (relationship arcs / flat relationships); category 5 (show, don't state); category 6 (the memorable physical detail, not a catalogue). |
| John Truby, *The Anatomy of Story* (ISBN 978-0865479937) | Category 3 & 4: the opponent must want the same deep thing (not just block the hero); the **cast web** built by contrast; the subplot character as a parallel problem resolved differently. |
| Robert McKee, *Story* (ISBN 978-0060391683) | Category 5: characterization vs. true character — depth is revealed by **choice under pressure**, not stated in exposition. |
| Ross Hartmann & Esther Chilton, *The Structure of Story* (ISBN 978-1735603810) | Background for category 1's fixes: the ghost/lie behind a missing need, and the subconscious architecture a flat want lacks. |
| Jason Georgiades & Rosita Grigaite, *7 Day Screenplay* (Udemy) — udemy.com/course/sevendayscreenplay | Background: the want / need / why / method / obstacle / stakes checklist used to spot what's missing. |

This skill mirrors the findings-table format of `swe/code-smell-audit` and pairs
with `writing/character-building` as its critique counterpart.
